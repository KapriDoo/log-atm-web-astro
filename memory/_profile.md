# log-atm-web-astro

## Stack

- **Lenguaje**: TypeScript (strict mode, Astro tsconfig)
- **Framework**: Astro 6.1.5 + React 19.2.5
- **Styling**: Tailwind CSS v4.2.2 (@tailwindcss/vite)
- **Animation**: GSAP 3.14.2, Motion 12.38.0
- **Icons**: Astro Icon + Lucide
- **Fonts**: Inter, Outfit (Fontsource)
- **Bundler**: Vite (via Astro)
- **Node**: >=22.12.0

## Convenciones

- **Commits**: Conventional Commits
- **Branches**: feature/, fix/
- **PR base**: main

## Notas del proyecto

- Componentes Astro con islands de React para interacciones client-side
- GSAP disponible para animaciones avanzadas
- Sharp para optimización de imágenes
