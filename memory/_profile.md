# log-atm-web-astro

## Stack

- **Lenguaje**: TypeScript (strict mode, Astro tsconfig)
- **Framework**: Astro 6.1.5 + React 19.2.5
- **Styling**: Tailwind CSS v4.2.2 (@tailwindcss/vite)
- **Animation**: GSAP 3.14.2, Motion 12.38.0
- **Icons**: Astro Icon + Lucide
- **Fonts**: Inter, Outfit (Fontsource)
- **Bundler**: Vite (via Astro)
- **Image processing**: Sharp 0.34.5
- **Sitemap**: @astrojs/sitemap
- **SVG optimizer**: SVGO 4.0.1
- **Node**: >=22.12.0
- **Deploy**: Docker (nginx + Brotli)
- **Site URL**: https://logatm.com

## Convenciones

- **Commits**: Conventional Commits (en inglés)
- **Branches**: feature/, fix/
- **PR base**: main
- **Código/comentarios**: español
- **Design tokens**: DESIGN.md (Google Stitch format)
- **Accesibilidad**: WCAG 2.2 AA mínimo
- **Performance**: Lighthouse ≥ 95
- **Animaciones**: prefers-reduced-motion obligatorio

## Notas del proyecto

- Componentes Astro con islands de React para interacciones client-side
- GSAP disponible para animaciones scroll-triggered
- Motion (Framer Motion) para islas React
- Sharp para optimización de imágenes
- Empresa: LOG ATM — "LOGÍSTICA A TU MEDIDA"
- Tono: Profesional cercano, latinoamericano
